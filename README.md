# TotalWar-Warhammer3-Analysis
# 《全面战争：战锤3》游戏系统拆解文档
> 一份基于底层数据表的硬核游戏拆解作品集，旨在展示系统策划的核心能力。

## 📖 项目简介
*   **游戏类型**：回合制战略 + 即时战术 (TBS + RTT)
*   **拆解时长**：约 3 个月
*   **分析时长**：500h+ (全系列)，涵盖双极难难度破局技巧
*   **文档体量**：100+ 页深度拆解报告

## 🎯 核心产出
*   **🔬 逆向硬核拆解**：**基于游戏底层数据表**，逆推了**命中判定、护甲减伤、冲击伤害与碰撞伤害结算**等核心战斗公式。
*   **🧠 方法论提炼**：提炼出 **"物理模拟 × 士气博弈 × 隐藏信息"** 三层设计框架，剖析《战锤3》如何将策略游戏升华为战争游戏。
*   **🚀 闭环重构能力**：基于拆解结论，独立设计了一套完整的 **"纳迦什DLC"派系机制**及**兽人Waaagh反比例士气机制**，形成从分析到重构的能力闭环。

## 🛠 技术栈与工具
*   **策划工具**：Excel, MATLAB, Xmind, Visio, Axure RP
*   **游戏数据库**：RPFM (用于读取和解包游戏 .pack 文件)
*   **编程辅助**：Python (数据翻译、清洗), Git (版本控制)
